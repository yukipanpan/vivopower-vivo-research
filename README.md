## 一句话结论
真正赚钱的澳洲电力业务（Kenshaw）在 FY2024 卖掉后基本成空壳；18 个月内三次换故事（能源 → XRP 加密财库 → AI 数据中心房东）；三年累计净亏 ~$83.8M，连续三年 going concern 警告。挪威 Mo i Rana 数据中心（$31M 既存 hosting 收入）是迄今唯一有实质现金流的资产，但控股%、AI 租户签约、会计处理都还未落到审计财报，待 FY2026 20-F 验证。
